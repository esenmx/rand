# Rand Example

A Flutter app demonstrating the [rand](https://pub.dev/packages/rand) package.

## Running the example

```bash
cd example
flutter run
```

## Features demonstrated

- **Numbers**: `integer()`, `float()`, `boolean()`, `latitude()`, `longitude()`
- **Identity**: `firstName()`, `lastName()`, `fullName()`, `alias()`, `city()`
- **Text**: `word()`, `words()`, `sentence()`
- **Cryptographic**: `id()`, `nonce()`, `password()`
- **Time**: `dateTime()`, `duration()`
- **Colors**: `color()`, `colorDark()`, `colorLight()`
- **Collections**: `element()`, `subSet()`
- **Sampling**: `sample()` with optional weights
- **Nullable**: `nullable()`
